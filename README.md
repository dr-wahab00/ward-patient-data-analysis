# Ward Patient Data Analysis

This project analyzes a simulated hospital ward dataset using Python Pandas.

## Clinical Questions Answered
- Average SpO2 in diabetic vs non-diabetic patients
- Diagnosis with highest average SBP
- High-risk patient identification
- Percentage of ICU patients who are diabetic
- Top 3 most severe patients
- Gender distribution in ICU admissions
- Average glucose in hypotensive patients
- Severity category distribution

## Skills Demonstrated
- Pandas Data Analysis
- Clinical Data Interpretation
- Feature Engineering (risk_flag, severity_score)
- GroupBy operations
- Filtering & Sorting

## Tools Used
- Python
- Pandas
- Jupyter Notebook
